# Assets

This folder contains screenshots and visual assets for the Eventure project.

## Screenshots

Add screenshots of your working website here. Suggested screenshots:

1. **Home Page** - Event list view (`event-list.png` or `home.png`)
2. **Create Event** - Form to create new events (`create-event.png`)
3. **Event Detail** - Individual event detail page (`event-detail.png`)
4. **Mobile View** - Responsive design on mobile (`mobile-view.png`)

## Naming Convention

- Use lowercase with hyphens: `event-list.png`
- Be descriptive: `home-page-events-grid.png`
- Include viewport info if needed: `mobile-view-events.png`

## Recommended Screenshots

To take good screenshots:

1. **Desktop View (1920x1080 or similar)**

   - Home page with events list
   - Create event form
   - Event detail page

2. **Tablet View (768px width)**

   - Responsive layout

3. **Mobile View (375px width)**
   - Mobile-optimized layout

## How to Add Screenshots

1. Take screenshots using:

   - Windows: `Win + Shift + S` (Snipping Tool)
   - Browser DevTools: Right-click → Inspect → Toggle device toolbar → Screenshot

2. Save screenshots in this folder with descriptive names

3. Update the main README.md to reference these screenshots

---

**Note:** Screenshots should be in PNG or JPG format. Keep file sizes reasonable (< 2MB each if possible).
